Simple Flask App

Aplikacja Dydaktyczna wyświetlająca imię i wiadomość w różnych formatach dla zajęć o Continuous Integration, Continuous Delivery i Continuous Deployment.

  • W projekcie wykorzystamy virtual environment, dla utworzenia hermetycznego środowisko dla aplikacji:

``` # tworzymy hermetyczne środowisko dla bibliotek aplikacji: $ python3 -m venv .venv

# aktywowanie hermetycznego środowiska $ source .venv/bin/activate $ pip install -r requirements.txt $ pip install -r test_requirements.txt

# zobacz $ pip list ```

Sprawdź: tutorial venv oraz biblioteki flask.

  • Uruchamianie applikacji:

``` # jako zwykły program $ python main.py

# albo: $ PYTHONPATH=. FLASK_APP=hello_world flask run ```

  • Uruchamianie testów (see: http://doc.pytest.org/en/latest/capture.html):

$ PYTHONPATH=. py.test $ PYTHONPATH=. py.test --verbose -s
